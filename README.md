# Iris

## Installation

### Prerequisites

- GHC (Glasgow Haskell Compiler): Recommended version 9.2.x or higher.
- Cabal: The standard build tool for Haskell.

### Build Steps

1. Build the project:
    ```bash
    cabal build --only-dependencies
    ```
2. Install:
    ```bash
    cabal install
    ```
