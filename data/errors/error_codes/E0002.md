A string literal was opened with `"` but never closed.

```iris
let x = "hello
```

```
error[E0002]: unterminated string
 --> main.iris:1:9
  |
1 | let x = "hello
  |         ^ string opened here, never closed
```

Every string literal must be closed with a matching `"` on the same line.
Multi-line strings are not supported.
