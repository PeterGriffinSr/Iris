Function parameters must be plain identifiers.

```iris
let add(1, 2) { 1 + 2 }
```

```
error[E0201]: expected parameter name
 --> main.iris:1:9
  |
1 | let add(1, 2) { 1 + 2 }
  |         ^ expected an identifier, not a number
= hint: function parameters must be plain identifiers
```

Parameter names follow the same rules as any identifier - they must
start with a letter or underscore and contain only letters, digits,
and underscores.

```iris
let add(a, b) { a + b }
```
