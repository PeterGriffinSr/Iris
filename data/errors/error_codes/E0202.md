The parameter list of a function declaration was not closed with `)`.

```iris
let add(a, b { a + b }
```

```
error[E0202]: expected ')' after parameter list
 --> main.iris:1:12
  |
1 | let add(a, b { a + b }
  |            ^ expected ')' here
```

Every `(` opening a parameter list must have a matching `)` before the
function body.

```iris
let add(a, b) { a + b }
```
