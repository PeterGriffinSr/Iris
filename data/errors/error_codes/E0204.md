A `let` value binding is missing the `=` between the name and its value.

```iris
let x 42
```

```
error[E0204]: expected '=' after name in let binding
 --> main.iris:1:7
  |
1 | let x 42
  |       ^ expected '=' here
```

Every `let` value binding requires `=` to separate the name from the value.

```iris
let x = 42
let y = x + 1
```

If you meant to declare a function, add a parameter list before the body.

```iris
let foo() { 42 }
```
