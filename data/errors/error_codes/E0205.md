An `if` expression is missing its condition.

```iris
if { 1 }
```

```
error[E0205]: expected condition after 'if'
 --> main.iris:1:4
  |
1 | if { 1 }
  |    ^ expected a condition here
```

An `if` expression requires a boolean condition between the keyword and
the then-block.

```iris
if true { 1 }
if x > 0 { x }
if a == b { "equal" } else { "not equal" }
```
