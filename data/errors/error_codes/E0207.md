The else-branch of an `if` expression must be wrapped in braces.

```iris
if true { 1 } else 2
```

```
error[E0207]: expected '{' after 'else'
 --> main.iris:1:20
  |
1 | if true { 1 } else 2
  |                    ^ expected '{' here
= hint: wrap the else-branch in braces: else { ... }
```

Both branches of an `if` expression must be blocks. For chained
conditions, nest another `if` inside the else block.

```iris
if true { 1 } else { 2 }

if x > 0 {
    "positive"
} else {
    if x < 0 { "negative" } else { "zero" }
}
```
