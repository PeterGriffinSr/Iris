A unary operator is missing the value it should apply to.

```iris
let x = -
```

```
error[E0209]: expected expression after unary '-'
 --> main.iris:1:10
  |
1 | let x = -
  |          ^ expected a value here
```

A unary operator must be immediately followed by a value.

```iris
let x = -42
let y = !true
let z = -(a + b)
```
