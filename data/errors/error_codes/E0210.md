A function call's argument list was not closed with `)`.

```iris
let x = fib(10
```

```
error[E0210]: expected ')' after argument list
 --> main.iris:1:15
  |
1 | let x = fib(10
  |               ^ expected ')' here
```

Every `(` opening an argument list must have a matching `)`.

```iris
let x = fib(10)
let y = add(1, 2)
```
