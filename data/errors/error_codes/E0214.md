A block must contain at least one expression.

```iris
let foo(x) {}
```

```
error[E0214]: empty block is not allowed
 --> main.iris:1:12
  |
1 | let foo(x) {}
  |            ^^ block contains no expressions
= hint: a block must contain at least one expression
```

Every block must have a value - its last expression is what the block
evaluates to. Add at least one expression inside the braces.

```iris
let foo(x) { x }
let one() { 1 }
if true { 1 }
```
