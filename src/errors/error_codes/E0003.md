A string literal contains a backslash escape that Iris does not recognise.

```iris
let x = "\q"
```

```
error[E0003]: unknown escape sequence '\q'
 --> main.iris:1:10
  |
1 | let x = "\q"
  |          ^^ unknown escape sequence
```

Valid escape sequences in Iris are:

| Sequence | Meaning        |
|----------|----------------|
| `\n`     | Newline        |
| `\t`     | Tab            |
| `\r`     | Carriage return|
| `\"`     | Double quote   |
| `\\`     | Backslash      |
