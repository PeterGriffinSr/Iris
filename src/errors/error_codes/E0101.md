The interpreter found the file but could not read its contents.

```
error[E0101]: could not read 'main.iris'
= hint: check that you have read permission on the file
```

The file may be owned by another user, have no read permission set, or be a
directory rather than a source file.

On Linux and macOS you can inspect and fix permissions with:

```sh
$ ls -l main.iris
$ chmod +r main.iris
```

On Windows, right-click the file, select Properties, and check the Security
tab to ensure your user account has read access.
