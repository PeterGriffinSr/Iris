A `let` binding must be followed by a name.

```iris
let = 42
```

```
error[E0200]: expected identifier after 'let'
 --> main.iris:1:5
  |
1 | let = 42
  |     ^ expected a name here
= hint: write: let <name> = <expr>  or  let <name>(...) { ... }
```

Every `let` must name what it is binding. The name must start with a
letter or underscore and contain only letters, digits, and underscores.
