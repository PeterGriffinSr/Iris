A function declaration must have a block body immediately after the parameter list.

```iris
let add(a, b) a + b
```

```
error[E0203]: expected '{' after parameter list
 --> main.iris:1:15
  |
1 | let add(a, b) a + b
  |               ^ expected '{' here
= hint: function body must be a block: let add(...) { ... }
```

Function bodies must always be wrapped in braces. If you want a simple
value binding without parameters, omit the parameter list entirely.

```iris
let add(a, b) { a + b }   # function
let result = a + b         # value binding
```
