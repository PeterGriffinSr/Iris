The then-branch of an `if` expression must be wrapped in braces.

```iris
if true 1
```

```
error[E0206]: expected '{' after if condition
 --> main.iris:1:9
  |
1 | if true 1
  |         ^ expected '{' here
= hint: wrap the then-branch in braces: if <cond> { ... }
```

Both branches of an `if` expression must be blocks.

```iris
if true { 1 }
if x > 0 { x } else { 0 }
```
