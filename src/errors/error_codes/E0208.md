A binary operator is missing its right-hand side.

```iris
let x = 1 +
```

```
error[E0208]: expected expression after '+'
 --> main.iris:1:12
  |
1 | let x = 1 +
  |            ^ expected a value here
```

Every binary operator requires a value on both sides.

```iris
let x = 1 + 2
let y = a * b
let z = 1 == 1 && 2 == 2
```
