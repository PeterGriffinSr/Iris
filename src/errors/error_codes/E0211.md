A grouped expression was opened with `(` but never closed.

```iris
let x = (1 + 2
```

```
error[E0211]: expected ')' to close grouped expression
 --> main.iris:1:15
  |
1 | let x = (1 + 2
  |               ^ expected ')' here
```

Every `(` used for grouping must have a matching `)`.

```iris
let x = (1 + 2)
let y = (a * b) + c
```
