A block was opened with `{` but never closed with `}`.

```iris
let foo(x) {
    x + 1
```

```
error[E0213]: expected '}' to close block
 --> main.iris:1:12
  |
1 | let foo(x) {
  |            ^ block opened here, never closed
```

Every `{` must have a matching `}`. Check that all nested blocks are
properly closed.

```iris
let foo(x) {
    if x > 0 {
        x
    } else {
        0
    }
}
```
