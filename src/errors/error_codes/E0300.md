A name was used that has no definition in the current scope.

```iris
let x = foo
```

```
error[E0300]: undefined name 'foo'
 --> main.iris:1:9
  |
1 | let x = foo
  |         ^^^ not defined in this scope
= hint: make sure 'foo' is defined before it is used
```

Every name must be bound by a `let` before it can be used. Check for
typos, or make sure the binding appears before the reference.

```iris
let foo = 42
let x = foo
```

At the top level, functions may refer to each other regardless of
order — mutual recursion is supported.

```iris
let ping(n) {
    if n == 0 { true } else { pong(n - 1) }
}

let pong(n) {
    if n == 0 { false } else { ping(n - 1) }
}
```
